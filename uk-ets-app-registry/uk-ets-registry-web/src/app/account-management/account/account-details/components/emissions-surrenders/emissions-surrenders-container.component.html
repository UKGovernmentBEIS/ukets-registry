Help with current surrender status
  • A: The number of allowances surrendered is greater than or equal to verified emissions
  • B: The number of allowances surrendered is lower than verified emissions
  • C: Verified emissions were not entered
  • Error: There was an error in the dynamic surrender status calculation. Send an email to etregistryhelp@environment-agency.gov.uk to report this error
  • Excluded: Operator is excluded from reporting obligations during this year
  • Not currently applicable: The emissions reporting period has not started for this operator
