Claim or assign a task

You can claim a task to manage any actions yourself, or assign it to another user to complete.

Select Tasks from the top menu.

Select the tasks you want to claim or assign.

If you’re claiming a task, you’ll be asked to enter a comment.

If you’re assigning a task, you’ll be asked to enter the name of the Authorised Representative you want to assign it to. A list of matching names will appear as you type. You’ll be asked to enter a comment.
