Adding or removing an account

Accounts from the same Account Holder are automatically added to the trusted account list (TAL).

Once an account is opened, you can add accounts from different Account Holders to the TAL.

To add or remove an account in the TAL:

  • select Accounts in the top menu
  • in the account list, select the account that you want to set up a trust for
  • select the Trusted Accounts menu item
  • select Request Update
  • select whether you want to add or remove an account
  • enter the account number for the account you want to add to TAL

A different Authorised Representative will need to approve the request. Once the request is approved, the account will be added to the TAL within 4 working days.
