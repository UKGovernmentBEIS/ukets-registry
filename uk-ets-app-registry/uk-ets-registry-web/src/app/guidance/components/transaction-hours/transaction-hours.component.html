Transaction hours and delays

Transactions on the UK Registry take place within set processing times.

Transaction rules for UK Registry actions

  • the UK Registry transaction window is Monday to Friday 10am to 4pm (UK time), excluding bank holidays in England and Wales
  • the Account Holder can set up transaction rules such that only transactions to accounts in the Trusted Account List (TAL) are permitted
Action Second approver required Processing time
Add or remove account from TAL Yes 4 working days
Add or amend an Authorised Representative to an account Registry Administrator Immediate (subject to UK working hours)
Change two-factor authentication device (FreeOTP) Registry Administrator Immediate (subject to UK working hours)
