UK Emissions Trading Registry guidance

Accessing the UK Registry

You must access the Registry using a desktop or laptop computer or a tablet. You will need access to a smartphone to use the FreeOTP Authenticator app that will allow you to securely sign in to the Registry.

Contents

Using FreeOTP Authenticator

The UK Registry uses the FreeOTP Authenticator app to generate single use two-factor authentication security codes that you will use to sign in and 'digitally sign' transactions within the UK Registry.

Full guidance can be found in the Registry FreeOTP Authenticator guide.
